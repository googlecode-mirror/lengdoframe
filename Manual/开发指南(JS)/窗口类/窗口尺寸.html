在使用窗口类的构造函数创建窗口时，可以提交窗口的宽度或高度参数来控制窗口的尺寸。在窗口创建完成后我们仍然可以使用以下参数来控制窗口的尺寸：

窗口类 - 成员函数： width(...)　 height(...)


1. 窗口尺寸 - 构造时设置宽度和高度
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id', null, {'width':400,'height':100} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
/script [演示]
通过以上的代码，可以创建出一个宽400px，高100px的窗口！ [ API：窗口类-窗口尺寸 ]

2. 窗口尺寸 - 创建后设置宽度和高度
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id');

wnd.create();
wnd.buttonAddDefault();

wnd.show();

alert("点击'确定'后设置宽度"); wnd.width(400);
alert("点击'确定'后设置高度"); wnd.height(100);
/script [演示]
通过以上的代码，可以创建出一个宽400px，高100px的窗口！ [ API：窗口类-窗口尺寸 ]

3. 窗口尺寸 - 窗口最大化
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id');

wnd.create();
wnd.buttonAddDefault();

wnd.show();

alert("点击'确定'后设置最大化"); wnd.max();
alert("点击'确定'后恢复最大化"); wnd.remax();
/script [演示]
通过以上的代码，可以设置和恢复窗口最大化！ [ API：窗口类-窗口尺寸 ]

1. API：窗口类-窗口尺寸
Wnd.prototype.width( width );
@params int width 宽度值，undefined表示返回宽度
@return int 返回当前或者上一个窗口宽度

Wnd.prototype.height( height );
@params int height int或'auto'表示设置高度，undefined表示返回高度
@return mix 返回当前或者上一个窗口客户区高度

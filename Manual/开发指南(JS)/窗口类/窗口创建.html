窗口创建，作为开发过程中最基础的一步，它既简单又复杂！合理的配置参数，可以创建出各种类型的窗口！


1. 窗口创建 - 默认窗口
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id' );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
/script [演示]
通过以上的代码，可以创建出一个默认的窗口！ [ API：窗口类-构造函数 ]

2. 窗口创建 - 固定宽度
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id', null, {'width':400} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
/script [演示]
通过以上的代码，可以创建出一个固定宽度的窗口！ [ API：窗口类-构造函数 ] - [ 窗口尺寸 ]

3. 窗口创建 - 窗口回调
script type="application/javascript"
var wnd = new Wnd( 'wnd-unique-id', {'ok':deal-unique-id} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();

function deal-unique-id(){
alert( 'You Click OK Button !' )
}
/script [演示]
在上面的代码中，我们演示的是当窗口的 '确定' 按钮被点击后窗口自动调用自定义函数！

X. 附录
1. API：窗口类-构造函数
new Wnd( id, callbacks, configs );
@params str id 窗口ID
@params obj callbacks 回调函数集
fun callbacks.ok 当点击确定，窗口消失后调用此函数
fun callbacks.okb 当点击确定，窗口消失前调用此函数 - 当返回false时，ok函数终止
fun callbacks.cannel 当点击取消，窗口消失后调用此函数
fun callbacks.cannelb 当点击取消，窗口消失前调用此函数 - 当返回false时，cannel函数终止
fun callbacks.hidden 当窗口隐藏时调用此函数
fun callbacks.complete 当窗口加载完成后调用此函数
@params obj configs 配置集
str configs.title 窗口标题
int configs.width 窗口宽度。默认：200。 注：窗口客户区宽度=窗口宽度-2
int configs.height 窗口客户区高度。默认：'auto'
int configs.overlay 窗口遮掩层透明度。false表示不显示，0-100表示透明度
int configs.titleact 窗口标题栏的操作按钮 000(最小化，最大化，关闭)，默认001. 注：前辍0省略
int configs.overflow 窗口溢出时滚动条 0000(scroll-x，scroll-y，hidden-x，hidden-y). 注：前辍0省略

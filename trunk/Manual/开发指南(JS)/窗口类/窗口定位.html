在一个窗口被创建时，系统会自动设置窗口的位置为当前可见区域的左上角！ 但很多时候这个位置并不是我们所希望的！不同通过使用下面的方法来改变一个窗口的位置......

窗口类 - 成员函数：moved(...)

1. 窗口定位 - 全居中
<script type="application/javascript">
var wnd = new Wnd( 'wnd-unique-id', null, {'width':400} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
wnd.moved();
</script> [演示]
通过以上的代码，可以创建出一个水平和垂直都居中的固定宽度的窗口！ [ API：窗口类-窗口定位 ]

2. 窗口定位 - 水平居中
<script type="application/javascript">
var wnd = new Wnd( 'wnd-unique-id', null, {'width':400} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
wnd.moved(null,100);
</script> [演示]
通过以上的代码，可以创建出一个距顶部100像素并水平居中的固定宽度的窗口！ [ API：窗口类-窗口定位 ]

3. 窗口定位 - 垂直居中
<script type="application/javascript">
var wnd = new Wnd( 'wnd-unique-id', null, {'width':400} );

wnd.create();
wnd.buttonAddDefault();

wnd.show();
wnd.moved(100);
</script> [演示]
通过以上的代码，可以创建出一个水平居中并距左边100像素的固定宽度的窗口！ [ API：窗口类-窗口定位 ]
